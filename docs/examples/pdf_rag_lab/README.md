# PDF RAG Lab

## Papel na arquitetura
Exemplo oficial de pipeline AI/LLM/RAG dentro da plataforma.

## Objetivo
Demonstrar um fluxo didático e funcional de:
- ingestão de PDF
- extração de texto
- chunking
- embeddings
- indexação vetorial
- retrieval
- montagem de contexto
- chamada ao LLM
- resposta final

## Regras
- SQL manual e editável, mas no template fixo da plataforma
- TableSpec obrigatório
- validators obrigatórios
- official readiness aplicável antes de ambiente official
- storage guard obrigatório
