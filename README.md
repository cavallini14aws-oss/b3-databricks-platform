# b3-platform

Base inicial da plataforma B3 no Databricks.

## Objetivo
Criar uma estrutura profissional, preparada para:
- bronze / silver / gold
- ingestão via tabelas e arquivos
- automação
- exploração
- MLOps
- LLMOps
- deploy controlado

## Estratégia
- Git como fonte de verdade
- estrutura compatível com Databricks
- contexto central por ambiente
- naming centralizado
- logging padronizado
