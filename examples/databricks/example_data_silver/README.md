# example_data_silver

Exemplo oficial de transformação Silver na plataforma Databricks.

## Quando usar
Use este exemplo para higienização, padronização, validação e enriquecimento de dados vindos da Bronze.

## O que este exemplo mostra
- leitura de bronze
- transformação controlada
- criação de tabela Silver com governança
- descrição campo a campo
- tags por tabela e coluna
- política de anonimização para campos sensíveis

## O que o desenvolvedor deve alterar
- regras de limpeza
- colunas de negócio
- enriquecimentos
- decisão de masking por campo
