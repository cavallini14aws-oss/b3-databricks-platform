# example_data_gold

Exemplo oficial de camada Gold na plataforma Databricks.

## Quando usar
Use este exemplo para tabelas consolidadas, agregadas e prontas para consumo analítico ou de produto.

## O que este exemplo mostra
- leitura da Silver
- agregação/regra de negócio
- criação de tabela Gold com governança
- descrição da tabela e das colunas
- tags de tabela e coluna

## Observação
Gold deve preferir reduzir exposição de PII. Quando houver PII, a decisão precisa ser explícita.
