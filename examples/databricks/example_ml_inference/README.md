# example_ml_inference

Exemplo oficial de inferência de Machine Learning na plataforma Databricks.

## Quando usar
Use este exemplo para scoring batch, inferência operacional e publicação de resultado de modelo.

## O que este exemplo mostra
- leitura de entrada
- saída de predição
- tabela de inferência com governança
- tags por tabela e coluna

## O que o desenvolvedor deve alterar
- origem de features
- carregamento real do modelo
- regra de predição
- formato da saída
