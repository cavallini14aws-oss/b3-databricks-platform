# example_rag_standard

Exemplo oficial de RAG padrão na plataforma Databricks.

## Quando usar
Use este exemplo para fluxos de ingestão documental, chunking, embeddings e recuperação sem trilha MLflow.

## O que este exemplo mostra
- pipeline RAG padrão
- tabela de documentos/chunks com governança
- tags de tabela e coluna
- política de masking para campos sensíveis dos documentos

## O que o desenvolvedor deve alterar
- origem dos documentos
- estratégia de chunking
- modelo de embeddings
- regras de recuperação
