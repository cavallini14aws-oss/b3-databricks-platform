# example_ml_training

Exemplo oficial de pipeline de treino de Machine Learning na plataforma Databricks.

## Quando usar
Use este exemplo para preparação de dataset, treino, avaliação e persistência de artefato/modelo.

## O que este exemplo mostra
- leitura de dataset
- treino simplificado
- saída técnica de tracking
- tabela de métricas/governança
- tags e política de masking quando necessário

## O que o desenvolvedor deve alterar
- dataset de entrada
- features
- algoritmo
- tracking/registry real
