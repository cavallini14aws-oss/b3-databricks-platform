# example_rag_mlflow

Exemplo oficial de RAG com trilha MLflow na plataforma Databricks.

## Quando usar
Use este exemplo quando o fluxo RAG exigir governança adicional de modelo, experiment tracking, serving ou trilha MLflow.

## O que este exemplo mostra
- pipeline RAG com variant mlflow
- tabela de chunks com governança
- ponto de integração com MLflow
- tags e política de masking

## O que o desenvolvedor deve alterar
- origem dos documentos
- estratégia de chunking
- modelo de embeddings
- integração real com MLflow/model serving
