# example_data_bronze

Exemplo oficial de ingestão Bronze na plataforma Databricks.

## Quando usar
Use este exemplo como base para pipelines de entrada bruta, recepção de arquivos, tabelas legadas ou cargas iniciais.

## O que este exemplo mostra
- estrutura mínima de pipeline
- criação de tabela com governança
- comentários de tabela e coluna
- tags de tabela
- tags coluna a coluna
- exemplo de política de anonimização para campos sensíveis

## O que o desenvolvedor deve alterar
- nome da tabela
- domínio de negócio
- lista de colunas
- regras de ingestão
- campos realmente sensíveis do projeto

## O que o desenvolvedor não deve alterar sem necessidade
- padrão de governança
- convenção de tags
- campos técnicos
- estratégia de classificação/masking
